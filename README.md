# phi-soc-module-b

EPPQ — Phi-SOC Module B: Emergence of Lie Algebra Structures from Relational SOC Dynamics

## Overview

This module implements the emergence of algebraic (Lie-like) structures from relational self-organized critical (SOC) dynamics within the EPPQ framework.

The goal is to show how algebraic structure can arise from relational relaxation processes.

## Pipeline

1. Generate relational graph (initial state)
2. Apply SOC relaxation dynamics
3. Extract relational operators
4. Identify emergent algebraic structure

## Project Structure

- `src/` — core implementation
- `scripts/` — execution scripts
- `experiments/` — experiment configurations/results
- `legacy/` — previous versions (not used in pipeline)

## How to run

```bash
python scripts/run_pipeline.py
